## Basic usage Astro
Use Id video or Src 
```html
<YoutubeEmbed
  id="Jvd-d4"
  title="Example"
></YoutubeEmbed>
```

## Basic usage typescript
Import component and liteYoutube function
```html
<head>
<script>
import { liteYoutubeIframe } from "youtube-lite-iframe"
</script>
</head>
```
```javascript
  liteYoutube({
    id: "vs-5674"
    title: "Example"
  })
```
